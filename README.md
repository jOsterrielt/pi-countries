COUNTRIES - SPA

🌍 Bienvenido a COUNTRIES, una Aplicación de Página Única construida con React, Redux, Node, Express y Sequelize. Explora una colección seleccionada de datos de países, incluyendo nombres, banderas, continentes, capitales y más.

🚀 Objetivos:

Aplicar conceptos fundamentales de React, Redux, Node, Express y Sequelize.
Implementar principios básicos de diseño UX/UI para una experiencia de usuario mejorada.
Reforzar y conectar conceptos aprendidos a lo largo de la carrera.
Adoptar mejores prácticas en desarrollo.
Aprender y practicar el flujo de trabajo de Git para colaboración.
Utilizar y practicar metodologías de testing.
🎨 Estilo y Diseño:

Utilizar técnicas esenciales de estilización para una interfaz estéticamente agradable.
Priorizar los principios de Diseño de Experiencia de Usuario (UX) y de Interfaz de Usuario (UI).
🔄 Flujo de Trabajo:

Adoptar Git para control de versiones y colaboración.
Demostrar habilidades en ramificación, solicitudes de extracción y desarrollo colaborativo.
🧪 Testing:

Integrar y practicar metodologías de testing para una calidad de código robusta.
Garantizar la fiabilidad de la aplicación mediante prácticas de testing exhaustivas.
🌐 Tecnologías Utilizadas:

React
Redux
Node.js
Express
Sequelize
HTML/CSS
Git/GitHub
👩‍💻 Cómo Ejecutar:

Clona el repositorio.
Instala las dependencias con npm install.
Ejecuta la aplicación con npm start.
¡Explora y disfruta la Aplicación de Página Única COUNTRIES!
🚧 Trabajo en Progreso:

Mejoras continuas y ampliación de funcionalidades.
Correcciones de errores y optimizaciones.

📚 Aprendizaje y Crecimiento:

Este proyecto es parte de un viaje de aprendizaje individual, con el objetivo de aplicar y consolidar conocimientos.
